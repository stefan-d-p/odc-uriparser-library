# UriParser for OutSystems Developer Cloud

Parses an Uri and returns

- AbsolutePath
- AbsoluteUri
- LocalPath
- Authority
- IsDefaultPort
- IsFile
- IsLoopback
- IsUnc
- PathAndQuery
- Segments
- Host
- Port
- Scheme
- Query
- Fragment
- UserInfo

This library has a single action `Parse` that takes an Uri as input parameter and returns a structure with the information above.
